# AI_RULE

AI가 새 책 활동을 만들 때 반드시 따라야 하는 교육 원칙과 생성 규칙을 관리한다.

## 책 유형
- `text-picture`: 글이 있는 그림책
- `low-text-picture`: 글이 적은 그림책
- `wordless-picture`: 글 없는 그림책

## 활동 영역 생성 원칙
- 모든 책에 같은 활동 영역 이름을 강제로 적용하지 않는다.
- `book.json`의 `bookType`과 `activitySections`를 기준으로 책 활동 홈을 구성한다.
- 글이 있는 그림책은 `읽기 활동`, `쓰기 활동`, `게임 활동`, `내 목소리`를 기본값으로 사용할 수 있다.
- 글이 적거나 글 없는 그림책은 `장면 상상하기`, `이야기 만들기`, `장면 말하기`, `문장 만들기`, `내 이야기 녹음하기`처럼 책 성격에 맞게 바꾼다.
- 학생 화면에는 `초급`, `고급`이라는 말을 사용하지 않는다. 활동은 쉬운 것에서 어려운 것 순서로 자연스럽게 배치한다.

## 저작권 보호
- 그림책 본문 이미지, 원화, 스캔 이미지를 사용하지 않는다.
- 책 제목, 작가, 출판사 정보만 표시한다.
- 활동 이미지는 자체 제작 아이콘, 말랑이 캐릭터, 추상 장면, 상징 그림, 교사가 직접 만든 이미지로 대체한다.
- 학생 화면과 생성 문구에서 `그림 읽기`, `그림을 보고`라는 표현을 사용하지 않는다.
- 대신 `장면을 떠올리고`, `상황을 상상하고`, `어울리는 말을 만들어 보고` 같은 표현을 사용한다.
